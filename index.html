2024.03 - 진행 중

Metaroad 그룹웨어 & 스케줄링/알람 시스템

Metaroad (직영/가맹 센터용 그룹웨어)
기술 스택 : Python 백엔드, 스케줄러 모듈(scheduler.py v2)
주요 기능 : 직영/가맹 센터 출결·송영·반복업무 통합 관리, 알람/배치 작업 자동화, 반복업무 스케줄링

직영/가맹 센터의 업무(출결, 송영, 반복업무)를 통합 관리하는 웹 기반 그룹웨어로, Python 백엔드와 스케줄러 모듈(scheduler.py v2)을 중심으로 알람·배치 작업·반복업무 자동화를 구현한 시스템입니다.

추가 설명:
Metaroad는 센터 운영에 필요한 출결·송영·반복업무 데이터를 하나의 웹 그룹웨어에서 통합 관리하는 실무형 프로젝트입니다. Python 기반 REST API(업무/출결/송영/알람 도메인)와 ORM(SQLAlchemy) 구조를 직접 설계하고, APScheduler 기반 스케줄러 엔진(scheduler_v2.py)으로 알람 발송 및 반복업무 롤오버(일·주·월·분기·반기·연간 규칙) 자동화를 구현했습니다. Celery·Redis 기반의 비동기 작업 처리로 대량 알림/배치를 분리하여 안정성을 높였으며, React 기반 관리자/센터 대시보드 일부 화면 개발에도 참여했습니다.
